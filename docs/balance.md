# Баланс Match-3

Этот документ фиксирует текущие допущения по балансу и начислению очков для 6 спринта.

## Базовые очки

- За очищенную фишку: `10`
- Бонус за линию из 4: `+40`
- Бонус за линию из 5 и более: `+120`
- Бонус за T/L-образную группу: `+90`

## Множители

- Множитель каскадного прохода: `chain` из цикла резолва (`x1`, `x2`, `x3`, ...)
- Множитель режима очков: `x1 | x2 | x3`
- Формула итогового начисления за проход: `Math.floor(baseWithShape * chain * scoreMode)`

## Спец-фишки

- При матче из 4 может создаваться спец-фишка `line` (строка или столбец).
- При матче 5+ или группе с пересечением может создаваться `bomb`.
- `line` при срабатывании очищает целую строку или столбец.
- `bomb` при срабатывании очищает область `3x3`.

## Параметры сложности уровня

- `boardSize`
- `durationSec`
- `goalValue`
- `tileKinds`
- `theme` (только визуальная настройка)

## Чеклист проверки

- После каждого ребаланса запускать:
  - `npx tsc --noEmit -p packages/client/tsconfig.json`
  - `npx jest -c packages/client/jest.config.js packages/client/src/game/match3/engine/core/core.test.ts --runInBand`
- Ручной smoke-check:
  - пересдача поля при тупике (deadlock reshuffle)
  - подсказка хода при бездействии
  - экран результата с вердиктом win/lose

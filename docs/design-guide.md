# Дизайн‑гайд Cosmic Match

## Базовые стили клиента:
  - `packages/client/src/shared/styles/normalize.pcss`
  - `packages/client/src/shared/styles/base.pcss`
  - `packages/client/src/shared/styles/landing.pcss` (лендинг).

## 1. Бренд и тональность

- Игра: космический match‑3 про сбор инопланетных сокровищ.
- Тон: дружелюбный, лёгкий, технологичный (UI — чистый, не перегруженный).

## 2. Цветовая палитра

- Основной текст: `#0F172A` (тёмный индиго).
- Фоновый светлый: `#F5F7FB`.
- Акцент 1 (фиолетовый): `#4F46E5`.
- Акцент 2 (зелёный): `#10B981`.
- Опасность: `#EF4444`.

### Темы лендинга

1. **Light Flat** — минималистичный, почти Notion:
   - Фон: `#F5F7FB`.
   - Карточки: `#FFFFFF` + лёгкая тень.
   - Акценты: `#4F46E5`, `#10B981`.

2. **Light 3D** — выпуклые элементы:
   - Градиентный фон (голубой → белый).
   - Карточки с сильной мягкой тенью, «пластиковые» фишки.

3. **Dark Neon** — космическая:
   - Фон: градиент `#1D243B` → `#020617` → `#000`.
   - Неон: `#38BDF8`, `#A855F7`, `#22C55E`.
   - Светлый текст `#E5E7FF` с лёгкой неоновой тенью.

## 3. Типографика

- Базовый шрифт: system‑ui, `14–16px` для body.
- Заголовки:
  - H1: 32–40px (hero).
  - H2: 24–28px (секции).
  - H3: 18–20px (карточки).

Все заголовки — без caps lock, только первая буква с заглавной.

## 4. Компоненты лендинга

### Header

- Высота 120px, sticky, blur‑фон.
- Логотип: созвездие фишек (круг с несколькими «звёздами»).
- Пункты меню:
  - Как играть (`#how-to-play`)
  - О проекте (`#about`)
  - Команда (`#team`)
  - Контакт (`#contact`)
  - Вход / Регистрация (роут на `/login` / якорь `#reg-login`)

### Hero

- Две колонки (текст + визуал) на десктопе, одна на мобиле.
- Визуал: схема поля 6×6 с подсвеченной линией «3 в ряд» и анимацией эффекта.

### Блоки

- **Как играть** — шаги геймплея match‑3.
- **О проекте** — преимущества (карусель).
- **Команда** — 5 карточек.
- **Обратная связь** — форма `mailto`.

Секции авторизации/профиля/форума/лидерборда визуально похожи на «Обратную связь».

## 5. Базовые отступы

- Максимальная ширина контента: 1200px (кроме полноширинных элементов).
- Внутренние отступы секций: 24–32px.
- Вертикальные отступы между секциями: 40–64px.

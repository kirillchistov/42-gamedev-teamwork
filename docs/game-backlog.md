# Backlog развития Cosmic Match

Документ хранит живой статус задач по игре.

Связанные документы:

- `docs/game-design.md` — что уже есть и как работает;
- `docs/game-task.md` — контракт по экранным состояниям;
- `docs/game-improvements-s7.md` — источник ближайших продуктовых улучшений.

## 1. Правила ведения

- Статусы: `todo`, `in_progress`, `done`, `icebox`.
- Каждая задача имеет владельца и ссылку на issue/PR.
- После закрытия задачи:
  - обновить статус здесь;
  - отразить изменение в `game-design.md` (раздел "Текущее ядро").

## 3. Экранные задачи

### 3.1 Состояние `done`

- Экран старта с настройками матча.
- Экран активной игры.
- Экран завершения с результатом и действиями.
- Кнопки перехода к форуму, лидерборду и профилю с `/game`.

### 3.2 Состояние `todo`

- Вывод альтернативных путей после поражения.
- Расширенный onboarding под новые механики (ходы, бустеры, цели).

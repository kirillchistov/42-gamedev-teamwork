## План доработок

### 6.3.1 — VFX при срыве матча (частицы + вспышка)
**Цель:** дать заметную обратную связь в момент очистки матча, не смешивая слой с логикой ядра.

| Файл | Изменения |
|------|-----------|
| `packages/client/src/game/match3/engine/matchFx.ts` | Новый модуль: частицы, радиальная вспышка, шаг симуляции, сброс. |
| `packages/client/src/game/match3/engine/renderer.ts` | Экспорт `boardLayout`; `renderBoard` / `pickCellAt` используют ту же геометрию. |
| `packages/client/src/game/match3/engine/bootstrap.ts` | Опциональный `fxCanvas`, `createMatchFx`, rAF-цикл, вызов из `flashMatches`, сброс в `resetIdle` / `destroy`. |
| `packages/client/src/game/match3/Match3Screen.tsx` | Второй canvas, передача `fxCanvas` в `createMatch3Game`. |
| `packages/client/src/game/match3/match3.pcss` | Стек canvas (`match3__board--stack`, `--board` / `--fx`). |

**Доп. настройка силы эффекта:** параметры в `matchFx.ts` (количество частиц, градиент вспышки, затухание).

---

### 6.3.2 — Оболочка страницы игры: полноэкран, компактная шапка, луна/солнце

**Цель:** больше полезной площади под игру, быстрый dark/light на странице игры, полноэкран (подключение Web API) и UX игрока.

| Файл | Изменения |
|------|-----------|
| `packages/client/src/utils/fullscreen.ts` | Новый модуль: enter/exit/toggle, webkit-префиксы, `addFullscreenChangeListener`. |
| `packages/client/src/pages/GamePage.tsx` | `ref` на корневую оболочку, `Header variant="game"`, горячая клавиша `F` (не в полях ввода). |
| `packages/client/src/components/Header/index.tsx` | `variant`, `fullscreenTargetRef`; на `/game` — компактные контролы, иконка полноэкранного режима, одна кнопка темы. |
| `packages/client/src/contexts/LandingThemeContext.tsx` | `LAST_LIGHT_THEME_KEY`, запись при светлой теме, `toggleColorMode()`. |
| `packages/client/src/shared/styles/variables.pcss` | `--header-height-game`, `--game-logo-icon`. |
| `packages/client/src/shared/styles/base.pcss` | `.landing-header--game`, `.landing-theme-toggle`, `.landing-fullscreen-btn`, `.game-page-shell:fullscreen`. |
| `packages/client/src/shared/styles/themes.pcss` | Стили переключателей в `dark-neon` для игровой шапки. |

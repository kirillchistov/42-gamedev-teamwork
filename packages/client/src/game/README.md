[] Что:  спец-фишки: “зачистка ряда” + “бомба (3x3)”; анимационные фазы highlight → clear → fall → refill.

[] Результат: Игрок получает доп фан со спецэффектами; Добавить простые визуальные эффекты для спец-фишек.

## 7. Поиск возможных ходов + пересдача при тупике
[] Что: Проверка “есть ли ход”; если нет — автоматический shuffle.
[] Как: добавить findPossibleMoves(board) (перебор соседних свапов + проверка матча);
после каждого resolve проверять наличие ходов; при пустом списке запускать анимированный shuffleBoard() до валидного состояния без стартовых матчей.
[] Результат: Игрок не застревает на “тупиковом” поле.

## 8. Подсказка хода по таймеру бездействия

[] Результат: легче начинающему игроку.
